C25OptimizationChallenge.breadth_first — Method
breadth_first(city; max_depth=5)

This function tries to find a good solution to the problem using a greedy approach with lookahead, using sequential computation of each of the car agendas.

C25OptimizationChallenge.breadth_first_parallel — Method
breadth_first_parallel(city; max_depth=5)

This function tries to find a good solution to the problem using a greedy approach with lookahead, using parallel computation of each of the car agendas.

C25OptimizationChallenge.repeat — Method
repeat(city, num_times, method, max_depth)

This function tries to find a good solution to the problem by repeatedly calling method and keeping the best solution found. This should result in improvements if method is randomized.

C25OptimizationChallenge.select_node — Method
select_node(current, candidates, graph, visited_nodes, visited_edges, distances, times)

This function selects the next node to traverse from a list of candidates by assigning each node a score and choosing the highest. The score is based on how good the best path starting with that node is.
